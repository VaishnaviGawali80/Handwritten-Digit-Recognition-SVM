# Handwritten Digit Recognition using SVM

## Project Overview

This project uses the Support Vector Machine (SVM) algorithm to recognize handwritten digits from the sklearn digits dataset.

## Technologies Used

* Python
* NumPy
* Matplotlib
* Scikit-Learn

## Algorithm

* Support Vector Machine (SVM)

## Dataset

* sklearn.datasets.load_digits()
